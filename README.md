### Requirement

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/abcdefathur/TiktokDownloader
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run 
   ```js
   node index
   ```
